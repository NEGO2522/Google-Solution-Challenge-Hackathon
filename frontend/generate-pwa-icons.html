VolunteerBridge — PWA Icon Generator

Opens once in a browser to produce the two required PNG icons.

192 × 192

Download pwa-192.png

512 × 512

Download pwa-512.png
After downloading:
1. Move both files into frontend/public/icons/
2. Run npm install then npm run build
3. You can delete generate-pwa-icons.html — it's a one-time helper.
