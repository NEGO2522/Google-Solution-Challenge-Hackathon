# Screenshots

Place your app screenshots in this folder with these exact filenames so the README references them correctly:

| Filename | What to capture |
|---|---|
| `01-dashboard.png` | The volunteer home dashboard showing stats, quick actions, profile strength |
| `02-live-map.png` | The live crisis map with issue pins, volunteer markers, and the side panel |
| `03-smart-match.png` | The Smart Match modal showing ranked volunteers with match scores |
| `04-issue-report.png` | The issue report form with the map pin and Gemini AI enhance button |

## How to take good screenshots

1. Run the app locally: `cd frontend && npm run dev`
2. Log in as a volunteer or admin (use the seed data)
3. Use browser fullscreen (F11) then take a screenshot at 1440px wide for best quality
4. On Windows: `Win + Shift + S` to snip a region
5. On Mac: `Cmd + Shift + 4` to select a region
6. Save as PNG with the filenames above

Once added, they will automatically appear in the README.
