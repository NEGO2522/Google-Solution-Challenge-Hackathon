# VolunteerBridge

> AI-powered volunteer matching for disaster relief and community crisis response.

## What it does

VolunteerBridge connects NGO admins with verified volunteers using a smart matching algorithm that scores volunteers by skills, proximity, trust score, and urgency level.

## Features

- 🔐 Auth — Volunteer & NGO Admin roles
- 🗺️ Live Map — Real-time crisis issue pins
- 🤖 Smart Match — AI scoring (skills + distance + trust + urgency)
- ⚙️ Admin Console — Manage issues, volunteers, approvals
- 📋 Task Tracking — Accept, start, complete with proof
- ⭐ Ratings & Trust Score — Reputation system for volunteers
- 🔔 Notifications — Real-time updates via Supabase
